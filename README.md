# 📊 Mobile Sales Dashboard (Power BI)

## 🚀 Project Overview
This project presents an interactive **Mobile Sales Dashboard** built using Power BI. It helps analyze sales performance, track trends, and compare year-over-year growth.

The dashboard is divided into multiple pages to provide detailed insights and better user experience.

---

## 📄 Report Pages

### 1️⃣ Dashboard
- Displays overall KPIs:
  - Total Sales
  - Total Quantity
  - Total Transactions
  - Average Price
- Includes slicers for Model, Payment Method, and Year
- Shows Month-To-Date (MTD) sales trend

## 🎯 Objectives
- Analyze sales, quantity, and transactions
- Track MTD performance
- Compare Year-over-Year (YoY) performance
- Identify top-performing segments
- Enable data-driven decision-making

---

## 📌 Key Features
- KPI Cards for quick insights
- Interactive slicers for dynamic filtering
- Time-based analysis (Daily, Monthly, Yearly)
- Year-over-Year comparison visuals
- Clean and user-friendly dashboard design

---

## 🛠️ Tools & Technologies
- Power BI
- DAX (Data Analysis Expressions)
- Power Query

---

## 📈 KPIs Explained
- **Total Sales** – Total revenue generated
- **Total Quantity** – Number of units sold
- **Total Transactions** – Number of orders
- **Average Price** – Average price per unit

---

## 📊 Insights
- Consistent growth trend in MTD sales
- Strong Year-over-Year performance improvement
- Seasonal trends visible across months
- Certain quarters contribute higher revenue
